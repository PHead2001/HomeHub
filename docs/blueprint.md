# **App Name**: HomeHub

## Core Features:

- Dashboard UI: Modern tablet dashboard with modules for each feature.
- Quick Actions: Quick-add buttons on the home dashboard for common actions (e.g., 'Add grocery,' 'Log maintenance').
- Maintenance Log AI Summaries: Use tool to summarize logs. Maintenance log summary dashboard element using LLM.
- Shared Grocery List: Grocery list with shared real-time editing capabilities.
- Pet Management Module: Track pet info (name, photo, vet records, food schedule), set reminders for pet care, chore tracking for pet care (walks, feedings, litter boxes, grooming)

## Style Guidelines:

- Primary color: Light cool-gray (#D3D3D3) for a modern, minimal feel.
- Background color: Very light cool-gray (#FAFAFA) for a clean backdrop.
- Accent color: Teal (#4DB6AC) for interactive elements and highlights.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Simple, modern icons for each module and feature, using a thin line style.
- Dashboard in a 2x8 grid layout optimized for a 16:9 display. Responsive design adapting to 9:16 layouts for phones and portrait tablets.
- Smooth sliding and tap-to-select transitions between modules. Minimal loading screens.
